Use The Radar On iPhone

  1. Run the Windows launcher and copy the printed phone URL, like http://192.168.x.x:8787.
  2. Open that URL in iPhone Safari.
  3. Your iPhone and computer must be on the same Wi-Fi.
  4. In Safari, tap Share, then Add to Home Screen.

Local manual events save only in the browser and device where you add them.

GitHub Pages (anywhere)

  1. Open https://<your-username>.github.io/socal-event-radar/ in Safari.
  2. Tap Share, then Add to Home Screen.
  3. Data refreshes every 6 hours automatically.

Local manual events save only in that browser/device — they don't sync between local and GitHub Pages.
